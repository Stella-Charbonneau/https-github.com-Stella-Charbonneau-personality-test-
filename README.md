For my final project, I created a personality quiz that determines what percentage of specific personaities a person posseses. The code does this by prompting the user with 13 different questions about how they would respond to different situations, storing each answer as either A, B, C or D. A, B, C and D are each correlated with a personality type, and once all answers have been entered, the code calculates the amount of times answers A, B, C and D were selected and returns the associate types and converted percentages to the user. There are no special requirments for running this code. Once the code is run, a question should be generated before another appears, with answers being stored and summed up until the personality type is printed in the terminal.
